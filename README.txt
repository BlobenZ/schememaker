Installation:
1) Make sure you have python installed
2) python -m pip install PIL

Usage:
scheme.json:
Replace team names (team1 = Left, team2 = Right)
Replace division 
Replace date
NOTE: Times can not be changed

Errors:
If program closes and no scheme was created then check following:
- Team names are spelled correctly (matches team logo)
- Team logo exists and follows name convention (Team name - TRI.png)
If none of the above works, contact BlobenZ :D
